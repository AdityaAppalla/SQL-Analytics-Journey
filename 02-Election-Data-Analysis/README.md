# Election Data Analysis using SQL

## Overview

This project analyzes election constituency data using SQL to study party performance, candidate rankings, constituency-wise results, and voting trends. The analysis was performed using MySQL with queries involving aggregation, filtering, joins, subqueries, and ranking-based insights.
---

## Dataset

The dataset includes:

- Candidate Name
- Party
- EVM Votes
- Postal Votes
- Total Votes
- Vote Percentage
- Constituency ID

---

## SQL Concepts Used

- SQL
- MySQL
- Aggregation Functions
- GROUP BY
- Filtering
- Joins
- Subqueries
- Data Analysis

---

## Analysis Performed 

### Query 1 Total votes received by each political party. 
### Query 2 Top 10 candidates with the highest total votes. 
### Query 3 Average vote percentage of each party. 
### Query 4 Total number of candidates from each party. 
### Query 5 Total votes recorded in each constituency. 
### Query 6 Top independent candidates based on total votes. 
### Query 7 Candidates with less than 1% vote share. 
### Query 8 Winning candidate in each constituency using joins and subqueries. 
### Query 9 Number of constituencies won by each party. 
### Query 10 Average total votes received by parties.

---

## Key Insights

- BJP received the highest total votes in the dataset.
- National parties dominated most constituency wins.
- Independent candidates had lower overall representation.
- Vote distribution differed heavily across constituencies.
